# The Collision Program

Five self-contained papers from a six-year research
program (2020-2026) studying the digit function
floor(br/p) and its collision structure.

## Papers

### A. The Collision Invariant (March 2026)

The collision invariant S measures bin coincidences in the
digit function floor(br/p). Four theorems: the gate width
theorem (exactly b-1 deranging multipliers), finite
determination (S depends only on p mod b^{l+1}), the
reflection identity (S(a) + S(m-a) = -1), and the half-group
theorem (every non-trivial diagonal slice splits the units
in half).

### B. The Collision Transform (March 2026)

The Fourier decomposition of S over Dirichlet characters.
The antisymmetry theorem restricts the centered transform to
odd characters. The centered sum F° converges at s=1 (no
Mertens term). Conditional penetration below s=1. The
neutrality theorem, perfect cancellation, and the base sum.

### C. The Collision Spectrum (March 2026)

The decomposition theorem: each Fourier coefficient factors
into a generalized Bernoulli number and a diagonal character
sum, encoding L-function special values exactly. The Parseval
moment identity gives an exact weighted second moment of
|L(1,chi)|^2. Conditional cross-moment in the critical strip.
Correlation decay. Class number connection at base 5.

### D. Partial Character Sums and the Collision Transform (April 2026, in preparation)

The bin character sum identity: the collision transform at any
modulus equals the sum of squared partial character sums over
the bins of the digit function. Connects the collision program
to the classical theory of Polya-Vinogradov and Burgess. Ghost
collision dominance at composites. The factoring wall. The
interleaving rank structure.

### E. The Collision Prime Number Theorem (April 2026, in preparation)

The centered collision deviations cancel across primes at
the PNT rate, with an effective constant (no Siegel zero
caveat for b >= 3). The collision periodic table: 40
integers encoding the collision fingerprint of every prime
via its last two digits. Antisymmetry S(a)+S(-a)=-1, proved.
L-value extraction from the table, universal for all bases.
